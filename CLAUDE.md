## 기획 출력 규칙
기획안 작성 시 반드시 아래 형식을 따를 것:
1. `plans/` 디렉토리에 `PLAN-{YYYYMMDD}-{feature-name}.md`로 저장
2. 각 태스크는 Codex가 독립적으로 실행할 수 있도록 분리
3. 태스크마다 아래 항목 필수 포함:
   - 파일: 수정/생성할 파일 경로
   - 설명: 구체적 구현 내용
   - 의존성: 선행 태스크 번호
   - 검증: 테스트 명령어 또는 확인 방법
